# 一、配置

## 生成配置文件

输入`karma init karma.conf.js`将会生成 js 配置文件

## 启动

输入`karma start karma.conf.js`将会启动测试，更多选项比如，`karma start karma.conf.js --log-level debug --single-run`，输入`karma start --help`可查看更多可选项
